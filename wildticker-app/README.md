Wild Ticker - Project Skeleton

This folder holds a minimal Android
project scaffold for Wild Ticker.
Open in Android Studio to finish
dependencies, API keys and build.

Quick start
1. Open this folder in
   Android Studio.
2. Add your API keys to a
   secure place (server or
   local encrypted file).
3. Build and run on device
   (overlay requires manual
   permission enabling).
